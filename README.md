makeStories, 创建在不同的人身上发生的一样的~~故事~~笑话.

### 运行

1. 注释掉上面和下面写着 "Comment all of these if you're not need to logging" 的代码 (如果你不想记录运行时间和内容. 如果你要, 记得更改 projectRootPath 为这个说明文件的绝对位置（本地）哦)

2. 更改布尔 "is" (江安为true，李丽丽为false)

3. 立刻运行 "Main.java"
